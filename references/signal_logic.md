# 信号聚合逻辑说明

6 大类信号（v2 新增多因子合成），最终加权汇总为一个总分 → 映射到走势倾向。

## 1. 当日涨跌（[-1, +1]）

```
+5% 以上    +1.0  强势
+1~5%       +0.5  小阳
-1~+1%      0     窄幅
-5~-1%      -0.5  小阴
-5% 以下    -1.0  重挫
```

## 2. 技术指标（原始 [-3, +3] → ×0.7 加权）

来自 `technical.analyze_signals()`。**注意原始分会乘 0.7 后才进入聚合**，避免技术指标占比过重。

| 规则 | 评分 |
|---|---|
| 均线多头排列 (MA5>10>20>60) | +1 |
| 均线空头排列 | -1 |
| 价 > MA20 / 价 < MA20 | ±0.3 |
| RSI ≥ 80 / ≤ 20 | ∓1（超买扣分，超卖加分） |
| RSI ≥ 70 / ≤ 30 | ∓0.5 |
| RSI 50-70 偏强 / 30-50 偏弱 | ±0.2 |
| 突破布林上轨 (BB% ≥ 1) | -0.3（强势但回吐风险） |
| 接近上轨 (0.8-1) | +0.3 |
| 跌破下轨 / 接近下轨 | +0.3 / -0.3 |
| MACD 金叉 / 死叉 | ±0.5 |
| MACD 柱放大 / 缩小 | ±0.2 |
| 放量上涨/下跌 (量比≥1.5 且涨跌≥2%) | ±0.5 |
| 缩量上涨 (量比<0.7) | +0.1（持续性弱） |

## 3. 资金流（[-1, +1]）

```
主力净流入 > 1 亿元   +1.0
        > 1000 万    +0.5
        < -1000 万   -0.5
        < -1 亿      -1.0
```

## 4. 龙虎榜（仅标记，不影响方向）

上榜本身=资金博弈剧烈，但方向中性（机构席位/游资席位的辨别需要更多规则，当前版本暂不加方向分）。

## 5. LLM 情感（[-1, +1]）

由 Claude API 输出原始分 [-5, +5]，乘以置信度系数（low=0.5 / medium=0.75 / high=1.0），再 / 5 归一化。

```
weighted = (llm_score / 5) * conf_mult
```

LLM 已做事件去重（同事件多家媒体 = 1 条）和重要性分级，所以**不需要**再做关键词词频。

## 6. 多因子合成（v2 新增，[-0.8, +0.8]）

4 大类因子各自打分 [-1, +1]，类别分 = 类别内因子均分；类别间按权重合成 → ×0.8 进 aggregator。

### 类别权重

| 类别 | 权重 | 数据源 | A 股 | 港股/美股 | ETF |
|---|---|---|---|---|---|
| **技术因子** | 35% | K 线 | ✓ | ✓ | ✓ |
| **风格因子** | 20% | `ak.stock_value_em` | ✓ | ✗ | ✗ |
| **基本面因子** | 25% | `ak.stock_financial_analysis_indicator` | ✓ | ✗ | ✗ |
| **相对强度** | 20% | peers + `ak.stock_zh_index_daily` | ✓ | △（仅 vs 行业） | ✗ |

缺失类别按比例归一化（实际权重 = w_i / Σw_available）。

### 各因子单独评分规则

**技术因子（5 项）**：
- 5 日动量：±8%/±3% → ±1.0/±0.5
- 20 日动量：±20%/±8% → ±1.0/±0.5
- 20 日波动率：≥85% / ≥65% / ≤20% 历史分位 → -0.5 / -0.2 / +0.3
- 量比(5/20日均量)：≥1.5 + 价升 → +0.7；≥1.5 + 价跌 → -0.7；≤0.7 → -0.2
- 布林位置 BB%：≥1 → -0.3（突破超买）；0.8-1 → +0.5；≤0 → +0.5；0-0.2 → -0.5

**风格因子（3 项，A 股）**：
- PE(TTM) 3 年分位：≤20% → +1.0；≤35% → +0.5；≥80% → -1.0；65-80% → -0.5
- PB 3 年分位：同 PE
- PEG：≤0.5 → +1.0；≤1 → +0.5；1-1.5 → 0；1.5-2.5 → -0.5；>2.5 → -1.0；<0 → -0.3

**基本面因子（5 项，A 股）**：
- ROE：≥15% → +1.0；≥8% → +0.5；3-8% → 0；0-3% → -0.5；<0 → -1.0
- 销售毛利率：≥40% → +1.0；≥25% → +0.5；15-25% → 0；5-15% → -0.5；<5% → -1.0
- 资产负债率：≤30% → +0.5；30-50% → +0.2；50-70% → -0.3；>70% → -0.8
- 净利润同比：≥50% → +1.0；≥20% → +0.7；5-20% → +0.3；-10~5% → 0；-30~-10% → -0.5；<-30% → -1.0
- 营收同比：≥30% → +1.0；≥10% → +0.5；0-10% → +0.1；-10~0% → -0.3；<-10% → -0.8

**相对强度（2 项）**：
- vs 行业当日 RS（个股涨跌% - peers 平均涨跌%）：≥3pct → +1.0；≥1pct → +0.5；-3~-1pct → -0.5；<-3pct → -1.0
- vs 上证 20 日 RS（个股 20 日收益% - 上证 20 日收益%）：≥15pct → +1.0；≥5pct → +0.5；-15~-5pct → -0.5；<-15pct → -1.0

### 因子有效性检验（v3 新增）

**目标**：识别噪声因子，避免无效因子稀释总分。

**方法 — 单股时序 IC + 滚动 IR**：

```
对每个可时序计算的因子 F：
  factor_series = F 在过去 ~700 个交易日的原始序列
  fwd_ret = close.shift(-5) / close - 1     # 5 日远期收益
  IC = spearman_corr(factor_series, fwd_ret)
  rolling_ic = factor_series.rolling(60).corr(fwd_ret)
  IR = rolling_ic.mean() / rolling_ic.std()
```

**评级**：

| 评级 | 条件 | 处置 |
|---|---|---|
| A | \|IC\| ≥ 0.10 + \|IR\| ≥ 1.0 | 全权（×1.0） |
| B | \|IC\| ≥ 0.06 + \|IR\| ≥ 0.5 | 全权（×1.0） |
| C | 仅一项通过阈值 | 半权（×0.5） |
| D | 都不达标 | 剔除（×0.0，不参与平均） |
| ? | 样本量不足 | 视为静态因子，原权重 |

**可验证因子**（8 项）：5d 动量、20d 动量、20d 波动率、量比、布林位置、PE 历史分位、PB 历史分位、vs 上证 20 日 RS

**静态因子**（7 项，跳过检验）：ROE、毛利率、资产负债率、净利同比、营收同比、PEG、vs 行业当日 RS

**为什么用时序 IC 而非横截面 IC**：
- 单股票分析没有横截面（cross-section）数据
- 时序 IC 检验"该因子在该股票上的预测力"，符合单股场景
- 限制：单股 IC 噪声大，IR 难以达到 0.5；故 C 级（半权）作为缓冲带，避免一刀切

**为什么用 |IC| 而非 IC（带符号）**：
- IC 符号在单股上不稳定（小样本噪声）
- 业界共识方向（动量看多、超买扣分）内置在打分逻辑
- 取 \|IC\| 评级 = 评估"因子与收益的关联强度"，方向沿用打分逻辑

**实测对预测稳健性的提升**（603893 瑞芯微，2026-05-27）：

| 模式 | 多因子合成 | 总信号 | 走势倾向 | 实际 5/28 |
|---|---|---|---|---|
| 不验证 | +0.51 | +2.28 | 看多 ↑↑（中高置信度） | -3.92% ❌ |
| **加 IC/IR 验证** | +0.32 | **+0.38** | **弱偏多 →↗（低置信度）** | -3.92% ⚠️ 预警 |

加上检验后，预测从"高置信度看多（错）"变成"低置信度弱偏多（更稳健）"。

## 总分映射

```
total ≥ 2     看多 ↑↑   置信度 中高
total ≥ 1     偏多 ↗    置信度 中
total ≥ 0.3   弱偏多    置信度 低
[-0.3, 0.3]   震荡 →    置信度 低
total ≤ -0.3  弱偏空
total ≤ -1    偏空 ↘
total ≤ -2    看空 ↓↓
```

## 局限

- 技术指标对短期（1-3 天）有效性有限；强趋势中 RSI 长期超买 ≠ 立刻回调
- 情感分析依赖 LLM 判断，对稀有事件/谣言的识别不稳定
- 没考虑：大盘 beta、板块轮动、行业政策、宏观事件、衍生品持仓
- 主要适用于流动性较好的中等市值股票，小盘股/ST/退市风险股结果不可信
